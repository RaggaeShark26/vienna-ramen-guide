# 🚀 Quick Add New Restaurant

## Copy This Template:

```json
{
  "name": "",
  "tags": ["", "", ""],
  "description": "",
  "address": ", Wien",
  "phone": "+43 1 ",
  "priceRange": "€€",
  "openingHours": "",
  "website": "",
  "instagram": ""
}
```

## Fill It Out:
1. **Name:** Restaurant name
2. **Tags:** 2-4 words (cozy, modern, authentic, etc.)
3. **Description:** 1-2 sentences about what's special
4. **Address:** Full Vienna address
5. **Phone:** +43 1 format
6. **Price:** € (cheap), €€ (medium), €€€ (expensive)
7. **Hours:** When they're open
8. **Website:** Full URL or leave empty ""
9. **Instagram:** Handle without @

## Add It:
1. Copy your filled template
2. Open `restaurants.json`
3. Add a comma after the last restaurant
4. Paste your new restaurant
5. Save!

**Done!** Your website updates automatically! 🍜
