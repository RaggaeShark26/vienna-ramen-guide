## 🚀 Quick Start - Update Your Website

### Step 1: Update Your Elementor Page
1. Go to your Elementor page editor
2. Find your **third HTML Widget** (the JavaScript one)
3. **Replace ALL the code** with the code from: [`integration-code.js`](https://github.com/RaggaeShark26/vienna-ramen-guide/blob/main/integration-code.js)
4. Click **"Update"**
5. **Done!** Your website now loads from GitHub! 🎉

### Your Data URL
Your restaurants are now served from:
```
https://raw.githubusercontent.com/RaggaeShark26/vienna-ramen-guide/main/restaurants.json
```

## 📱 Managing Restaurants (Super Easy!)

### ⚡ Add New Restaurant (2 minutes)
1. **Use the template:** [quick-add-template.md](https://github.com/RaggaeShark26/vienna-ramen-guide/blob/main/quick-add-template.md)
2. **Fill it out** with restaurant details
3. **Add to data:** [Edit restaurants.json](https://github.com/RaggaeShark26/vienna-ramen-guide/edit/main/restaurants.json)
4. **Save** - Your website updates automatically!

### ✏️ Edit Existing Restaurant
1. **Open:** [restaurants.json](https://github.com/RaggaeShark26/vienna-ramen-guide/edit/main/restaurants.json)
2. **Find** the restaurant you want to edit
3. **Change** any field
4. **Save** - Done!
